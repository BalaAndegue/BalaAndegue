## <img src="https://i.pinimg.com/originals/3f/7e/4e/3f7e4eff7c96e9fe4b8b4b1ff3f7bdb5.gif" width="5%"> About Me

<img align="right" src="https://github.com/7oSkaaa/7oSkaaa/blob/main/Images/Right_Side.gif?raw=true" width="28%">

👋 **I'm Lionnel Bala Andegue**, a **Systems Engineer** and **Full-Stack Developer** who builds software from the ground up — from raw TCP sockets and Linux kernel interfaces to React frontends and ML pipelines.

🔧 I write **C++20** and **Rust** for performance-critical systems, design **Reactor-pattern** servers with `epoll`, manage thread pools, and think in terms of cache lines, syscalls, and lock-free data structures.

🧠 I build **AI/ML models** (TensorFlow, PyTorch, Scikit-Learn) and **full-stack applications** (React, Django, Spring Boot).

📊 I monitor everything — **Prometheus** scrapes, **Grafana** dashboards, distributed traces. If it runs in prod, I observe it.

🐧 **Linux is my native environment** — I live in `gdb`, `valgrind`, `perf`, `strace`, `htop`, and the POSIX API.

🏆 **Hackathon winner** · **AI competition finalist** · **Event organizer** · Open source builder.

🌍 Based in **Cameroon** · Open to remote collaborations worldwide.

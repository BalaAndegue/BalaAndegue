## About Me

<img align="right" src="https://github.com/7oSkaaa/7oSkaaa/blob/main/Images/Right_Side.gif?raw=true" width="28%">

**Lionnel Bala Andegue** — Systems Engineer and Full-Stack Developer who builds software from the ground up, from raw TCP sockets and Linux kernel interfaces to React frontends and ML pipelines.

I write **C++20** and **Rust** for performance-critical systems, design **Reactor-pattern** servers with `epoll`, manage thread pools, and think in terms of cache lines, syscalls, and lock-free data structures.

I build **AI/ML models** with TensorFlow, PyTorch and Scikit-Learn, and **full-stack applications** with React, Django and Spring Boot.

I monitor everything — **Prometheus** scrapes, **Grafana** dashboards, distributed traces. If it runs in production, it gets observed.

**Linux is my native environment** — `gdb`, `valgrind`, `perf`, `strace`, `htop`, POSIX API daily.

Hackathon winner · AI competition finalist · Event organizer · Open source builder.

**Based in Cameroon** — Open to remote collaborations worldwide.
